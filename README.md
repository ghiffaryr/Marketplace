# Requirements

- Python 3.8 or above
- Flask

# Installation

- Go to specified directory which the app will be installed
- Create virtual environment for your python

`python3 -m venv venv` or `virtualenv venv`

- Clone the project from repository

`git clone [git url]`

- Go to project directory

`cd [project url]`

- Activate yout virtual environment

`source ../venv/bin/activate`

- Install the dependencies

`pip install -r requirements.txt`

- Start the server

`python3 run.py`

- You should now be able to reach your the application by visiting your hostname in your browser, for example

http://127.0.0.1:5000
